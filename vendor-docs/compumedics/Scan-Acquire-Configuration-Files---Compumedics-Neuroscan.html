Scan Acquire Configuration Files

Scan Acquire can be configured to work with a number of amplifiers and type of caps. Below you will find the different configurations for each of the amplifiers and the type of caps.

The configurations for the SynAmps 2/RT and the NuAmps are for Quik-caps that are specific to the amplifier (i.e. no adapter is required and the caps can be connected directly to the amplifier). The other configurations (listed under General) are for Quik-caps used for either the SynAmps 1 system, or for non-Neuroscan amplifiers.

In the zip-archives you will find a *.AST and *.3DD file for the listed configuration. For some of the configurations you will also find the corresponding *.DAT file.

File types

  • *.AST: setup file for use with Scan Acquire; not compatible with Curry acquisition
  • *.3DD: file containing 3D coordinates, created and saved using Scan 3DSpace. Note the 3DD files open in 3DSpace, and are not ASCII files. The .3DD files may also be used in Curry.
  • *.DAT: ASCII versions of the .3DD files. Note not all 3DD files have a corresponding ASCII version. An ASCII version may be created by opening the .3DD file in 3DSpace, and then saving as an ASCII (.DAT) file. The ASCII files may also be used in Curry.

Please note these files are for Scan users, and do not contain any Curry acquisition configurations.
